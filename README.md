## 💻 Getting Started <a name="getting-started"></a>

- To get a local copy up and running, follow these steps.

### Prerequisites

- In order to run this project you need:

Example command:

```sh
1.use a browser
2.use cable internet
```
 
### Setup

- Clone this repository to your desired folder:

Example commands:

```sh
  git https://github.com/Arch-Noize/to-do-list.git
  cd to-do-list
```

### Install

- Install VS code, Node.js and git on your local computer.

### Usage

- In order to run this project, clone this repository to your PC and open a terminal, then use:
```sh
npm start
```

### Run tests

If you are facing linter errors, use:

- For HTML:

```
npx hint .
```

- For CSS:

```
npx stylelint "**/*.{css,scss}"
```

- For Javascript:

```
npx eslint .
```
